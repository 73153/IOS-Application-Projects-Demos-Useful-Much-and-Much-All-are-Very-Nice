Detailed Description

The protocol is implemented by the PTSSpringboard to be informed of user-interactions by a single PTSSpringboardItem. This is the onyl purpose of this protocol. You should not edit it, unless you want to alter the PTSSpringboard classes behaviour.

- (void) springboardDidBeginDraging: (PTSSpringboardItem *)  item

A PTSSpringboardItem informs the delegate, that it did begin dragging (a touching event was detected). This method is only called, when the item has been marked movable by the data source.
