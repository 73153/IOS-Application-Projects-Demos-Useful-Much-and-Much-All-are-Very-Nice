Detailed Description

The PTSSpringBoard is used to display a list of user-interactable items in a view that resembles the iOS Homescreen. The items are dynamically arranged on the view and, if necessary, on multiple pages.

This class implements the behaviour of the PTSSpringBoard (a resizing behaviour, item-order, drawing and event handling) and uses delegation, to report user-interaction and to load content. The springboard needs a data-source which implements the PTSSpringBoardDataSource protocol. The data source provides the content and information on the basic behaviour of the springboard.

User-Interaction triggered events are reported to the delegate of the springboard, using the PTSSpringBoardDelegate protocol. The protocol can be used to implement actions, that are triggered when PTSSpringBoardItems are selected, moved or deleted by the user.
